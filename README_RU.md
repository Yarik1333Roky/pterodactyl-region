# Главная страница «pterodactyl-region»

![Full Screen showcase](https://github.com/Yarik1333Roky/pterodactyl-region/blob/ea810ec1420f95a27790dc907043a9ebe643ac1f/pictures/full_screen.png?raw=true)
![Logo Image](https://github.com/Yarik1333Roky/pterodactyl-region/blob/ea810ec1420f95a27790dc907043a9ebe643ac1f/pictures/regions_show.png?raw=true)

**pterodactyl-region** — это *бесплатный модуль*. Его основная задача — показывать, где в данный момент расположен ваш сервер.

## Полная настройка

![Admin Settings](https://github.com/Yarik1333Roky/pterodactyl-region/blob/main/pictures/admin_settings.png?raw=true)

Доступна *полная настройка* с использованием API. Поддерживаются:

* DNS Google, DNS Cloudflare
* ipapi.is, ipapi.co, ipwho.is, geoiplookup.io

## Установка

Используйте ***данную ссылку*** для установки модуля в среду Pterodactyl:

```bash
bash <(curl -sL https://exeyarikus.info/pterodactyl-region/install)
```

Запускайте этот скрипт **только с правами root**.
